4. Atividade realizada fora do plano (se couber)
14)      Descrição

15)      Comentários



a)       

Aquisição de ferramentas ETL.

A solução exigiu a aquisição de ferramenta ETL. Foi necessário especificar e apoiar a CPL durante todo o processo de contratação. Situação: atividade concluída.

b)      

Ser o principal responsável pela Contratação de Infra-Estrutura de Produção.

A expectativa inicial era que a Setec fornecesse a infra-estrutura de TI, diretamente ou através de terceiros. O Projeto acabou assumindo a maior parcela dos trabalhos referentes ao processo de contratação, elaborando estudos, justificativas e relatórios técnicos. Situação: atividade concluída.

c)       

Contratação de serviços da Reuters.

A Semag demandou a contratação dos serviços, o que exigiu a participação no processo de contratação. Situação: atividade concluída.

d)     A

Administração contratos.

Foi necessário administrar os contratos da Reuters e STK, verificando se os serviços estavam sendo prestados adequadamente. Situação: atividade em desenvolvimento.

e)       

Especificação do software de Análise Investigativa

Foi feita a especificação do software para o SGI em 2005.

f)        

Especificação de grande parte das funcionalidades da fase 2.

Diversos componentes foram especificados pela própria equipe, tais como Sistemas Internos (Matriz de Risco, Radar, Processus e Fiscalis), dados do SIEST e SIDOR,  Cadastro de Pessoas. Situação: atividade concluída.

g)      

Elaboração de Material de Treinamento para uso do Modelo de Dados do Síntese e construção de relatórios.

Foi necessário elaborar material específico para ensinar os usuários a compreender o modelo de dados e, assim, poderem desenvolver relatórios. Situação: atividade em desenvolvimento.

h)      

Depuração de erros.

Não pudemos contar com apoio dos usuários para depurar as funcionalidades do Síntese. Parte da equipe do projeto foi alocada para esta finalidade. Situação: atividade em desenvolvimento.

i)        

Elaboração de Relatórios para facilitar o uso.

Para facilitar o uso do Síntese, está sendo desenvolvido uma série de relatórios. Situação: atividade em desenvolvimento.

j)        

Desenvolvimento de sistema para a Semag, com o objetivo de calcular os índices que serão utilizados pelo Síntese para  certificar se estão corretos os valores transferidos pela União aos Estados/Municípios.

A Semag tinha dificuldades para calcular estes índices. Sem estes índices, o Síntese não tinha como analisar se os valores transferidos pela União aos Estados/Municípios estavam corretos. Por isto, desenvolveu-se esta funcionalidade. Situação: atividade em desenvolvimento desde outubro de 2005.
